# Mapy wiki

Folder trzyma mapy relacji miedzy artykulami i dzialami.

## Jak odpalic mape

1. Otworz plik mapa-wiki.md.
2. W VS Code uzyj podgladu Markdown lub rozszerzenia Mermaid.
3. Na GitHub Pages mapa renderuje sie automatycznie przez skrypt osadzony na dole pliku mapa-wiki.md.
4. Po dodaniu nowych hasel aktualizuj graf i sprawdz podglad po wdrozeniu strony.
